# 🚀 Modern Interactive Full-Stack Portfolio

A high-end, responsive portfolio website featuring **Glassmorphism** design, an interactive **Neural-Network Particle Background**, and a functional **Serverless Backend**.

## 🔗 Live Demo
**[https://bhavindama.github.io/my-portfolio/](https://bhavindama.github.io/my-portfolio/)**

---

## ✨ Key Features
* **Neural-Core Background:** A custom HTML5 Canvas particle system with mouse-proximity logic and physics-based drift.
* **Full-Stack Contact Logic:** Integrated with **Formspree** using AJAX/Fetch API to handle real-time form submissions without page reloads.
* **Academic Analytics:** Visualized academic performance (SPI/CGPA) using a synchronized SVG line graph and dynamic column charts.
* **Glassmorphism UI:** Modern frosted-glass aesthetic with high-fidelity blur and border-lighting effects.
* **Cloud-Optimized Media:** Project demonstrations powered by Cloudinary-hosted MP4s for lightning-fast performance.
* **Mobile Architecture:** Fully responsive layout designed to maintain visual integrity across all device resolutions.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Custom Variables/Flexbox), JavaScript (ES6+).
* **Interactivity:** HTML5 Canvas API (Neural Particle Physics).
* **Communication:** Formspree API (Serverless form handling) & AJAX Fetch.
* **Media Hosting:** Cloudinary (Content Delivery Network).
* **Typography:** Plus Jakarta Sans & Poppins.



---

## 📂 Project Structure
* `index.html` - Semantic structure, meta-tag optimization, and SVG layouts.
* `portfolio.css` - Advanced styling, glassmorphism definitions, and keyframe animations.
* `portfolio.js` - Neural particle engine, splash exit logic, and asynchronous form handling.
* `resume.pdf` - Professional credentials for recruiters.

## 🚀 Local Setup & Deployment
1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/bhavindama/my-portfolio.git](https://github.com/bhavindama/my-portfolio.git)
    ```
2.  **Configuration:**
    * Open `index.html` and replace the Formspree endpoint in the `<form action="...">` attribute with your unique Form ID.
    * Ensure your profile picture is named `bhavin.jpg` in the root directory.
3.  **Launch:**
    * Open `index.html` in any modern web browser to view the local build.
